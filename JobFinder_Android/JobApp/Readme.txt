Creating Android Launcher Icons
-------------------------------

The resource folder of each customer contains launcher icons for the app. 
This icons are stored in the following folders:

-drawable-hdpi
-drawable-ldpi
-drawable-mdpi
-drawable-xhdpi

The icons can be created online with the Android Asset Studio site (Option Launcher Icons).

http://android-ui-utils.googlecode.com/hg/asset-studio/dist/index.html
